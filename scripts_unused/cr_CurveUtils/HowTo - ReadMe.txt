Firstly thanks for your interest, 

This Maya MEL script package is created for easy creation and editing of animation controls and control shapes. All my daily controls creation and editing tools compiled into one tool.

For suggestions and info please feel free to contact me on my website
If you'd like to get more involved please visit our website creaturerigs.com .

Installation: 
===============================================
	please close your current Maya session:
	=======================================

	Copy the all mel files from "scripts" into your Maya scripts folder found here:
	
	C:\user\xxxxx\Documents\maya\2012\scripts
	
	Launch new Maya session and execute following command to launch curveUtil UI:
	
	cr_curveUtil();
